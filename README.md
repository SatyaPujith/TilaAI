## ğŸš€ Quick Start

### Prerequisites

- **Node.js** 18+
- **Raindrop CLI** - `npm install -g @liquidmetal-ai/raindrop`

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/tila-ai.git
cd tila-ai

# Install dependencies
npm install
cd raindrop-backend && npm install && cd ..
```

### Environment Configuration

Create `.env.local`:

```env
# AI Configuration
VITE_GEMINI_API_KEY=your_gemini_api_key

# Voice AI (ElevenLabs)
VITE_ELEVENLABS_API_KEY=your_elevenlabs_api_key
VITE_ELEVENLABS_AGENT_ID=your_agent_id

# Backend API
VITE_API_URL=https://tila-ai-api.01kbs9qyfs1fh8dnwn4k7g8mak.lmapp.run
```

### Run Locally

```bash
# Terminal 1 - Frontend
npm run dev

# Terminal 2 - Backend
cd raindrop-backend
raindrop build deploy --start
```
